# 教案 Markdown 转 Word 工具

这是一个使用 Python 编写的教案 `Markdown -> Word` 工具。

这一次不再重新绘制表格，而是直接基于 `教案模板.docx` 填充内容，因此能最大程度保持原模板版式不变，尽量做到和模板一模一样。

## 文件说明

- `md2word.py`：主程序，读取 Markdown 后直接填充 `教案模板.docx`
- `lesson_plan_template.md`：Markdown 模板
- `requirements.txt`：依赖文件
- `教案模板.docx`：原始 Word 模板

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方式

### 1）复制 Markdown 模板

```bash
copy lesson_plan_template.md 我的教案.md
```

### 2）填写内容

严格按照模板字段填写。

### 3）生成 Word

```bash
python md2word.py 我的教案.md
```

### 4）指定输出文件

```bash
python md2word.py 我的教案.md -o 我的教案.docx
```

### 5）指定模板文件

```bash
python md2word.py 我的教案.md -t 教案模板.docx -o 我的教案.docx
```

## 说明

- 程序默认使用当前目录下的 `教案模板.docx`
- 生成时直接保留模板中的标题样式、表格布局、分页和线框
- 如果你还觉得和模板有差别，通常只剩“填入哪个单元格”或“标题文字格式”这种小问题，可以继续微调
